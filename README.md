# Cafe Barcode - Restaurant Website

**Cafe Barcode** is a restaurant website built using **ASP.NET** for the backend and standard frontend technologies including **HTML5, CSS, and JavaScript**. The platform allows users to explore available food items, add favorites, and interact with the restaurant, while admins can manage the website content and user accounts efficiently.

---

## Features

### User Features
- User registration and login  
- Browse available food items on the homepage  
- View About section  
- Leave and read reviews  
- Contact section to reach out to the restaurant  
- Add food items to favorites  

### Admin Features
- Admin login only (no registration)  
- Add new food items with details: name, price, quantity, description, and image  
- Update existing food items  
- Delete food items  
- Manage user accounts (delete users)  

---

## Tech Stack

**Frontend:**  
- HTML5  
- CSS3  
- JavaScript  

**Backend:**  
- ASP.NET (C#)  

**Database:**  
- SQL Server

**Other:**  
- Authentication for admin and user accounts  
- CRUD functionality for food items
